Python Haqqında Məlumat

Python sadə sintaksisə malikdir və yeni başlayanlar üçün çox münasib proqramlaşdırma dilidir. Aşağıda Python ilə yazılmış bəzi sadə proqram nümunələri verilmişdir:

1. Ədədin cüt və ya tək olduğunu tapmaq

num = int(input("Ədədi daxil et: "))

if num % 2 == 0:
    print("Bu ədəd cütdür.")
else:
    print("Bu ədəd təkdir.")

2. Ədədin rəqəmlərinin cəmini tapmaq

num = int(input("Ədədi daxil et: "))
cem = 0

while num > 0:
    cem += num % 10
    num //= 10

print("Rəqəmlərin cəmi:", cem)

3. Fibonacci ədədləri

n = int(input("Neçə Fibonacci ədədi göstərilsin? "))

a, b = 0, 1
for i in range(n):
    print(a)
    a, b = b, a + b

4. Faktorial hesablayan proqram

def faktorial(n):
    if n == 0 or n == 1:
        return 1
    return n * faktorial(n - 1)

num = int(input("Ədədi daxil et: "))
print("Faktorial:", faktorial(num))

5. Sözün palindrom olub-olmadığını yoxlamaq

soz = input("Sözü daxil et: ")

if soz == soz[::-1]:
    print("Bu söz palindromdur.")
else:
    print("Bu söz palindrom deyil.")
